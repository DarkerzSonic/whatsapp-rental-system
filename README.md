# WhatsApp Rental Management System

Web Form which allow owners to enter rental details and send it through WhatsApp Web.

## Frameworks

Front-end: React + MUI

Back-end: Node + Express

## Project Setup
### Frontend Setup

#### Install Node Dependencies

```
npm install
```

#### Run React Website

```
npm start
```

### Backend Setup

#### Install Node Dependencies

```
npm install
```

#### Run Express Server

```
nodemon server.js
```
